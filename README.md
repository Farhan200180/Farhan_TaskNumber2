# BMI Calculator

## Description

A Python program that calculates the Body Mass Index (BMI) based on the user's weight and height.

## Features

* Calculates BMI
* Classifies BMI as Underweight, Normal Weight, Overweight, or Obese
* Easy to use

## Technologies Used

* Python

## How to Run

1. Run `bmi_calculator.py`
2. Enter your weight in kilograms.
3. Enter your height in meters.
4. View your BMI and category.
